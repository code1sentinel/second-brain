# Resources

Reference material organized by **topic** — not by project or area.

## What belongs here

- Notes on a topic you find interesting or useful
- Summaries of books, articles, courses, videos
- How-to guides and cheat sheets
- Anything you might want to retrieve later

## What does NOT belong here

- Material tied to a specific active project → `01-projects`
- Your personal responsibilities → `02-areas`

## Folder convention

```
03-resources/
└── topic-name/
    ├── README.md       ← overview / MOC (Map of Content)
    └── note-title.md
```
