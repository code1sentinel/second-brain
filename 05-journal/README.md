# Journal

Timestamped reflections — daily captures and weekly reviews.

## Structure

```
05-journal/
├── daily/      ← quick daily notes (YYYY-MM-DD.md)
└── weekly/     ← weekly review (YYYY-Wxx.md)
```

## Daily note (use template: 06-templates/daily-note.md)

- What happened today?
- What did I learn?
- What's on my mind?
- Top 3 for tomorrow

## Weekly review checklist

- [ ] Clear inbox
- [ ] Review active projects — any stuck? any done?
- [ ] Review areas — any standard slipping?
- [ ] Capture lessons learned this week
- [ ] Set top priorities for next week
