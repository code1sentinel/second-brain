# Areas

An **area** is a sphere of activity with a standard to maintain over time — no finish line.

## What belongs here

- Ongoing responsibilities: health, finances, relationships, career
- Standards you want to uphold indefinitely
- Things you regularly review but never "complete"

## Sub-areas

| Folder | Description |
|--------|-------------|
| [health](./health/) | Fitness, nutrition, sleep, mental health |
| [finances](./finances/) | Budget, investments, taxes |
| [learning](./learning/) | Skills, courses, reading list |
| [work](./work/) | Career goals, performance, professional network |

## Adding a new area

1. Create a sub-folder with a clear name.
2. Add a `README.md` describing the standard you want to maintain.
3. Link it in the table above.
