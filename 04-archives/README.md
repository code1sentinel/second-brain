# Archives

Completed projects and inactive material — kept for reference, not for action.

## What belongs here

- Finished projects (move the whole project folder here)
- Old area content that's no longer active
- Resources you've outgrown but don't want to delete

## Why archive instead of delete?

Context from past work is often useful later. Searching through archives has surfaced
the right answer more than once. Keep it; storage is cheap.

## Convention

Move folders here as-is. Prefix the folder name with the completion date:

```
04-archives/
└── 2026-06-project-name/
```
